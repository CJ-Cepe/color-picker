Introduction

Welcome to HueMore, your go-to color picker extension for Google Chrome. With a single click, explore and capture colors from any webpage. HueMore supports HEX, RGB, HSL, and CMYK color codes, making it a versatile tool for all your color needs. Enjoy the simplicity and efficiency of color picking with HueMore!

Security and Privacy Measures

  • HueMore does not access, collect, or modify any information on any site you visit.
  • HueMore operates under manifest version 3, which only grants storage permissions. This is solely used for storing the list of colors that users have collected.

How to Use

Selecting a Color

  1. Open the extension by clicking the HueMore icon.
  2. Click the 'Pick Color' button to activate the eyedropper tool.
  3. Point to the desired pixel and click to obtain its color codes.

Canceling picking color (while in color pick mode)

  • Option 1: Press the Escape key. This will immediately exit the color pick mode.
  • Option 2: Click on any pixel to select its color, then delete this color from your color history. This action is simply to exit from color pick mode.

Copying Color Codes

  1. Select a color using the 'Pick Color' button or by clicking on a previously selected color from the list on the right.
  2. To copy the code, just click the color code value and it will automatically copy it to your clipboard. Ready for pasting.

Viewing Previous Colors

  1. To view a previously selected color, simply click on the color you want from the list on the right side of the extension.
  2. Once clicked, its color code will be displayed, ready for you to copy or apply as needed.

Exporting/Downloading the Color List

  1. Click the 'Export/Download' button (represented by a download icon) to download your color list.
  2. This will generate a text file containing all the colors in your list, each with their equivalent HEX, HSL, RGB, and CMYK codes.
  3. The colors in the text file are sorted by the time they were added.

Deleting a Color from the History

  1. Click on the color you wish to delete.
  2. Click the "Delete" button (represented by a trash icon).
  3. The currently displayed color will be deleted from your history.

Clearing the Color History

  1. Click the "Clear" button (represented by a broom icon) to delete all colors from your history.

Troubleshooting

If you're experiencing issues with HueMore, please try the following steps:

  • Reload the Web Page: Refresh your browser page.
  • Restart the Browser: If refreshing the page doesn't work, try completely closing and reopening your browser.
  • Check for Updates: Make sure you have the latest version of HueMore installed. You can check for updates in the Chrome Web Store.
  • Toggle the Extension: Try turning the extension off and back on again. You can do this in the Extensions menu (chrome://extensions/).
  • Comment in the Feedback Section: If you're still having trouble, please comment your issue in the feedback section on the extension's page in the Chrome Web Store.

Other Important Notes

About the Color History (List)

  • The color list on the right side of the extension is sorted by the latest picked colors, arranged from left to right, top to bottom.
  • If you pick a pixel and its color is already in the list, it won't be added again to avoid duplication. However, it will still be selected, and its corresponding color code will be displayed.

Limitations

  • This extension was developed and tested using the latest version of the Chrome browser. Compatibility with older versions has not yet been verified.
  • The tool utilizes the EyeDropper API, which is currently classified as an ‚Äúexperimental technology‚Äù. Therefore, its behavior may change in future updates of the Chrome browser.
